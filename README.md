A Full Todo Application (Basic operations) developed using MERN Stack.
For Styling, Used Tailwind CSS [Ignore UI :)]

Features : 
- User Authentication
- Each user have their own List of tasks.
- Have implemented Create, Edit, Delete & Completed functionality.
- Frontend filtering for All, Completed and Incomplete Tasks.

Note:
add .env in Backend folder.
Variables needed are : JWT_SECRET, MONGO_URI & PORT
